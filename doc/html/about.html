Preccsについて

Preccsとは

PreccsはProtocol code generator based on Regular Expressions and ConCurrent processeSの略で，正規表現と並行プロセスにもとづいて記述され た通信プロトコルの仕様からC言語のプログラムを自動的に出力する通信プロ トコルコンパイラです．Preccs を利用することで，簡単に通信プロトコルを 記述することができ，さらにプロトコルを処理するコードが自動的に生成され ますから，実際に通信プロトコルの動作を動かして確認することもできます．

Preccsの特徴

Preccsは通信プロトコルを簡単に記述できるように設計されているので，ある 程度，プログラミングの経験がある人ならば，すぐにでも通信プロトコルを実 装することができます．さらに，Preccsコンパイラによって実際にプロトコル の処理を行うCのコードが生成されるので，実際に動かして動作を確認するこ とができます．Preccsの主な特徴は以下のとおりです．

  • 簡単な文法
  • 並行プロセスにもとづく処理手順の記述
  • 拡張正規表現によるメッセージ形式の定義
  • 正規表現パタンマッチ機能
  • PreccsコンパイラによるCコードの自動生成
  • インラインC

未踏ソフトウェア創造事業
